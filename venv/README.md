# AI Resume Parser

An automated system that extracts information from resumes using NLP.

## Features
- Resume upload (PDF)
- Extract name, email, phone
- Extract skills
- Detect education
- Detect experience
- Candidate dashboard
- Search candidates by skill

## Technologies Used
Python
Flask
spaCy (NLP)
PDFPlumber
SQLite
Bootstrap

## How to Run

1. Clone repository
2. Install dependencies

pip install -r requirements.txt

3. Run application

python app.py

4. Open browser

http://127.0.0.1:5000

## Project Purpose

This project demonstrates backend development and NLP-based resume parsing similar to Applicant Tracking Systems (ATS) used by HR platforms.
